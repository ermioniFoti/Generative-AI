# Generative AI: 3D Surface Reconstruction with SDFs

A deep learning project implemented in **PyTorch** to perform 3D surface reconstruction from point clouds. The system utilizes neural networks to predict **Signed Distance Functions (SDFs)**, enabling the transition from discrete point data to continuous 3D surfaces.

This project was developed as part of the Generative AI course (Spring 2025) at the Technical University of Crete.


## 📌 Project Overview

Implemented a PyTorch neural network to predict signed distance functions (SDFs) for 3D surface reconstruction from point clouds, including forward passes and training/validation.



Due to the lack of the original assignment documentation, this repository focuses on the core architectural implementation and the training pipeline.
